🏭 Architecture terrain réelle
Base de travail v4 — à adapter avec tes références exactes
V4 PRO

⚡ INVT Variateur PV

  • Commande marche / arrêt / sens arrière ajoutée.
  • Lecture statistiques : Vdc, Vac, Hz, A, kW, kWh.
  • Paramètres modifiables uniquement en maintenance.
  • Défaut tension DC basse bloquant au démarrage.

🔴 Salmson forage

  • Coffret conservé.
  • Sécurité manque d’eau raccordée sur bornes 21/22.
  • Pas de capteur niveau bas/haut dans le forage.
  • Commande marche/arrêt sous conditions de sécurité.

💧 Wilo surpresseur

  • Capteur pression déjà présent.
  • Capteur débit à ajouter pour m³/h et volume cumulé.
  • Paramètres Wilo affichés dans le module Paramètres.
  • Surveillance pression mini/maxi et marche sans débit.

🌊 Réservoir principal 10 000 L

  • Capteur niveau d’eau à ajouter.
  • Seuils bas / très bas / plein.
  • Alarme niveau bas bloquante pour le surpresseur.
  • Suivi volume estimé en litres.

🌿 Rain Bird

  • Accès natif via programmateur + module WiFi LNK/LNK2 si compatible.
  • Intégration directe possible surtout via passerelle type Home Assistant.
  • Le dashboard peut superviser et lancer des zones via API backend.
  • Le programme interne Rain Bird reste à confirmer selon module exact.
🚨 Stratégie alertes professionnelles
Ce qui doit bloquer automatiquement les pompes
⛔ Manque d’eau Salmson 21/22 : arrêt forage immédiat + interdiction redémarrage.
⛔ Réservoir principal très bas : arrêt ou blocage surpresseur Wilo.
⚠️ Pression Wilo anormale : alerte pression basse/haute, recherche fuite ou vanne fermée.
⚠️ Débit nul pompe en marche : suspicion marche à sec, vanne fermée ou capteur HS.
⚠️ INVT tension DC basse : pas de démarrage si solaire insuffisant.
